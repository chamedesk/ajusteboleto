VERSÃO WEB - AJUSTE DE FOTO BOLETO

Arquivos:
- app.py
- requirements.txt
- Procfile
- templates/index.html
- static/style.css

Como publicar no Render:
1. Crie uma conta no Render.
2. Clique em New + > Web Service.
3. Envie esta pasta para um repositório GitHub.
4. Conecte o repositório no Render.
5. O Render vai ler:
   - requirements.txt
   - Procfile
6. Depois de publicado, o sistema abrirá por link.

Uso:
- envie o PDF
